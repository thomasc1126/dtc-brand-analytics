# DTC Brand Analytics

End-to-end analytics engineering portfolio project: ingestion, dbt modeling, and BI dashboards for a direct-to-consumer brand's e-commerce, advertising, and retail data.

## Project Structure

```
ingestion/          — Python scripts for data cleaning and BigQuery loading
dbt_project/        — dbt models (staging → intermediate → marts)
docs/               — Project documentation
```
